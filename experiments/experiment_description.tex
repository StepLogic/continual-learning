\section{Evaluation Environment}

We evaluate continual RL agents on the \textbf{MetaWorld ML10 benchmark}, a suite of 10 robotic manipulation tasks using a Sawyer robot arm with 4-dimensional continuous action space. Agents face \textbf{catastrophic forgetting} as tasks switch every 400,000 environment steps.

\section{Training Protocol}

\begin{itemize}
    \item \textbf{Total steps}: 4,000,000 per seed
    \item \textbf{Steps per task}: 400,000 (cycles through all 10 tasks 4 times)
    \item \textbf{Action space}: $[-1, 1]^4$ continuous end-effector control
    \item \textbf{Algorithm}: Soft Actor-Critic (SAC) with twin Q-functions
    \item \textbf{Replay buffer}: 1,000,000 transitions
    \item \textbf{Batch size}: 256
    \item \textbf{Discount} ($\gamma$): 0.99
    \item \textbf{Target smoothing} ($\tau$): 0.005
\end{itemize}

\section{Metrics}

Performance is evaluated using standard continual RL metrics:
\begin{itemize}
    \item \textbf{Average Return}: Mean episode reward across all tasks
    \item \textbf{Forgetting Measure}: Performance drop on previous tasks after learning new ones
    \item \textbf{Forward Transfer}: Improvement on new tasks from prior learning
\end{itemize}
